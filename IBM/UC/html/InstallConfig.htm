NAME

InstallConfig - helper class (sub-class of Config) for Install module, loads and stores configuration info
for install.

SYNOPSIS

use IBM::UC::UCConfig::InstallConfig;

################# # class methods # #################

 $ob = InstallConfig->new($ini);
   where:
     $ini=configuration file (optional parameter)
 $ob = InstallConfig->createFromHash($cfg);
   where:
     $cfg=hash reference of parameter name/value pairs

####################### # object data methods # #######################

### get versions ###

see base class Config

### set versions ###

see base class Config

######################## # other object methods # ########################

see base class Config

DESCRIPTION

The InstallConfig class is a helper class for the Install module. It stores configuration information (parameter name/value pairs) for use by its encapsulating class. Initialisation of the config object can be via a configuration file (initFromConfigFile()) or via a hash map reference (initFromConfig($hashRef)).
