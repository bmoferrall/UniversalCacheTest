NAME

TestBedConfig - helper class (sub-class of Config) for TestBed module, loads and stores configuration info
for TestBed instances.

SYNOPSIS

use IBM::UC::UCConfig::TestBedConfig;

################# # class methods # #################

 $ob = TestBedConfig->new($ini);
   where:
     $ini=configuration file (optional parameter)
 $ob = TestBedConfig->createFromHash($cfg);
   where:
     $cfg=hash reference of parameter name/value pairs

####################### # object data methods # #######################

### get versions ###

see base class Config

### set versions ###

see base class Config

######################## # other object methods # ########################

see base class Config

DESCRIPTION

The TestBedConfig class is a helper class for the TestBed module. It stores configuration information (parameter name/value pairs) for use by its encapsulating class. Initialisation of the config object can be via a configuration file (initFromConfigFile()) or via a hash map reference (initFromConfig($hashRef)).
